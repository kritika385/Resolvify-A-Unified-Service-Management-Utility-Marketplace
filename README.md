# 🚀 Resolvify: A Unified Service Management & Utility Marketplace

**Resolvify** is a high-performance Java-based ecosystem engineered to bridge the gap between service providers and end-users. It serves as a centralized middleware that automates service discovery, simplifies administrative workflows, and provides real-time diagnostic capabilities.

---

### 🏗️ System Architecture & Logic

*   **Intelligent Discovery Engine:** Implements a secure authentication layer using mobile-number-based verification to fetch real-time account data from the relational database.
*   **Dynamic Pricing Engine:** Developed custom computational logic to automatically categorize accounts—distinguishing between **Domestic and Commercial** tiers—and apply tiered billing calculations without manual intervention.
*   **Service Marketplace Integration:** A comprehensive directory that allows for the discovery and booking of professional service providers through a unified interface.
*   **Real-time Diagnostics:** Features an integrated Network Speed Test module to monitor connectivity and infrastructure health directly from the application.
*   **Automated Grievance Redressal:** A streamlined support ticket system utilizing the **JavaMail API** for instant notification triggers and rapid resolution workflows.

---

### 🛠️ Technical Tech Stack

*   **Core Logic:** Java (J2SE)
*   **Platform:** Apache NetBeans
*   **Backend Database:** MySQL (Relational Data Persistence and Complex Querying)
*   **Integration APIs:** JavaMail API for automated communication
